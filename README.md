

# 🧪 Projeto QA - Teste de Carrinho de Compras

## 📌 Objetivo
Este projeto tem como objetivo simular testes manuais em um fluxo de e-commerce, validando o funcionamento do carrinho de compras e processo de finalização de pedido.

---

## 📚 Escopo de Testes

Foram realizados testes nas seguintes funcionalidades:

- Adição de produtos ao carrinho
- Alteração de quantidade de itens
- Validação de dados cadastrais
- Processo de pagamento
- Finalização de compra

---

## 🧪 Tipos de Teste Aplicados

- ✅ Teste Funcional  
- ✅ Teste Exploratório  
- ✅ Teste de Regressão  
- ✅ Teste de Smoke  

---

## 🛠 Técnicas de Teste Utilizadas

- Partição de Equivalência  
- Análise de Valor Limite  
- Tabela de Decisão  

---

## 🐞 Bug Encontrado

### 🔴 Duplicação de pedidos ao finalizar compra

**Descrição:**  
Ao clicar duas vezes rapidamente no botão "Finalizar Compra", o sistema gera pedidos duplicados.

**Severidade:** Crítica  
**Prioridade:** Alta  

**Impacto:**  
- Possível cobrança duplicada  
- Problemas financeiros  
- Impacto na experiência do usuário
